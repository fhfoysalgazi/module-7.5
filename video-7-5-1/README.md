# CSS Pseudo-classes  
Read the article--> [link](https://www.w3schools.com/css/css_pseudo_classes.asp)

__Style an element when a user moves the mouse over it__  
```
h3:hover{
    color
    background color
}
hover means mouse over.
```
__Style visited and unvisited links differently__  

```
a:visited{
    color
}
```

__Style an element when it gets focus__  
```
input:focus{
    color
    outline
    border
}
```
__Style an element that is the first child of its parent__  
```
li:last-child{
    color
}
li:nth-child(2n){
    color
}
li:nth-child(2n+1){
    color
}
li:first-child{
    color
}
```
